## ⚙️ How It Works

1. 🧩 Scan your codebase → parses functions & classes
2. ✍️ AI enhances docstrings for clarity
3. 📘 Generates Markdown files (`index.md`, `api.md`)
4. 🎨 Renders them with MkDocs Material
